# Ejemplos

Los mismos ejemplos vistos en clase, pero reescritos por mi, ya que Lino los deja desordenados.

- [Ejemplo 1](./Ejemplo%201.cpp): sobre la definición del arreglo y conceptos básicos del mismo.
- [Ejemplo 2](./Ejemplo%202.cpp): ejemplo de un programa que calcula la media y la desviación estándar utilizando arreglos y bucles.
- [Ejemplo 3](./Ejemplo%203.cpp): Ejemplo de un programa que crea un arreglo aleatorio y brinda el número máximo y mínimo del mismo, usando bucles para identificarlos.
- [Ejemplo 4](./Ejemplo%204.cpp): Un programa que calcula la hipotenusa de un triángulo rectángulo por medio de una función y que verifica que dos catetos y una hipotenusa forman un triángulo rectángulo cumpliendo el Teorema de Pitágoras, por medio de una función.
- [Ejemplo 5](./Ejemplo%205.cpp): Un programa que crea un arreglo, lo llena de valores aleatorios y busca un valor dado en el arreglo. Todo utilizando funciones.
- [Ejemplo 6](./Ejemplo%206.cpp): Un programa que recrea el juego de _piedra, papel o tijera_, utilizando `enum` para identificar los estados.
- [Ejemplo 7](./Ejemplo%207.cpp): Explicación básica de qué es un puntero.
