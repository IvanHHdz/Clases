# Simulación de circuitos

Este directorio contiene simulaciones de circuitos, principalmente aquellos que aparecen en la guía de la Ing. Lurwin. Esto con el objetivo de verificar las respuestas.

Se han recreado las guías en un solo archivo, para poder ubicarlos mejor.

Las simulaciones son hechas para ser simuladas haciendo uso de QUCS-S. Para más información de dicho simulador, consulte la página oficial [aquí](https://ra3xdh.github.io/).
